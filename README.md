# Arduino Projects Repository

## 🇺🇿 O‘zbekcha

Ushbu repository **Arduino asosidagi elektronika va IoT loyihalari** uchun yaratilgan. Bu yerda turli xil sensorlar, modullar va mikrokontrollerlar yordamida yaratilgan tajriba va amaliy loyihalar joylashtiriladi.

### 📌 Repository maqsadi

* Arduino asosidagi loyihalarni yig‘ish va ulashish
* Elektronika va dasturlashni o‘rganayotganlar uchun namunalar taqdim etish
* Sensorlar va modullar bilan ishlash bo‘yicha amaliy misollar berish

### 📂 Loyihalar tarkibi

Repository ichida quyidagi turdagi loyihalarni topishingiz mumkin:

* 🌡 **Sensor loyihalari**

  * DHT11 / DHT22 (harorat va namlik)
  * PIR Motion Sensor
  * Ultrasonic Distance Sensor

* 💡 **IoT loyihalari**

  * ESP32 / ESP8266 WiFi loyihalari
  * Web interface orqali boshqaruv

* 🔐 **RFID va xavfsizlik loyihalari**

  * RFID Access Control
  * Admin card tizimi

* ⚙ **Avtomatlashtirish loyihalari**

  * Motor boshqarish
  * LED indikatsiya tizimlari
  * Relay modul boshqaruvi

### 🛠 Ishlatilgan texnologiyalar

* Arduino Uno / Nano / Mega
* ESP32 / ESP8266
* C++ (Arduino IDE)
* Turli xil sensor va modullar

### 🚀 Qanday foydalanish mumkin

1. Repositoryni clone qiling:

```
git clone gh repo clone inoyatulla-dev/Arduino.git
```

2. Arduino IDE orqali `.ino` faylni oching
3. Kerakli kutubxonalarni o‘rnating
4. Kodni Arduino yoki ESP qurilmaga yuklang

### 🤝 Hissa qo‘shish

Agar siz ham Arduino loyihalari bilan ishlayotgan bo‘lsangiz, pull request yuborishingiz mumkin.

### 📜 Litsenziya

Ushbu loyiha **open-source** bo‘lib, ta’limiy maqsadlarda foydalanish mumkin.

## 🇬🇧 English

This repository contains **Arduino-based electronics and IoT projects**. It includes experiments and practical implementations using different sensors, modules, and microcontrollers.

### 📌 Purpose of the repository

* Collect and share Arduino projects
* Provide examples for learning electronics and programming
* Demonstrate practical use of sensors and modules

### 📂 Project categories

You can find the following types of projects in this repository:

* 🌡 **Sensor Projects**

  * DHT11 / DHT22 (temperature and humidity)
  * PIR Motion Sensor
  * Ultrasonic Distance Sensor

* 💡 **IoT Projects**

  * ESP32 / ESP8266 WiFi projects
  * Web-based control interfaces

* 🔐 **RFID & Security Projects**

  * RFID access control
  * Admin card system

* ⚙ **Automation Projects**

  * Motor control
  * LED indicator systems
  * Relay module control

### 🛠 Technologies used

* Arduino Uno / Nano / Mega
* ESP32 / ESP8266
* C++ (Arduino IDE)
* Various sensors and modules

### 🚀 How to use

1. Clone the repository:

```
git clone gh repo clone inoyatulla-dev/Arduino.git
```

2. Open `.ino` file using Arduino IDE
3. Install required libraries
4. Upload the code to your Arduino or ESP board
